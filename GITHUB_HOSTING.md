# GitHub Pages Hosting

## Koraki:
1. Ustvari novo GitHub repozitorij (npr. "ribic-law-website")
2. Naloži vse datoteke v repozitorij:
   ```bash
   cd "/Users/miha/Desktop/Cloude Code Test"
   git init
   git add .
   git commit -m "Initial commit - Leon Ribič website"
   git branch -M main
   git remote add origin https://github.com/TVOJ_USERNAME/ribic-law-website.git
   git push -u origin main
   ```
3. V GitHub repozitoriju pojdi na Settings > Pages
4. Izberi "main" branch in shrani
5. Tvoja stran bo na: https://TVOJ_USERNAME.github.io/ribic-law-website/

- ✅ Brezplačno
- ✅ SSL certifikat (HTTPS)
- ✅ Lahko dodaš custom domeno (ribiclaw.com)
